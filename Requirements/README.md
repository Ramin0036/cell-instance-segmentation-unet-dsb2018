## 📋 Requirements

Before running this project, ensure that the following software and libraries are installed.

### Software

| Software | Version |
|----------|---------|
| Python | **3.10.0** |
| pip | **26.1.2** |
| Jupyter Notebook / JupyterLab | Latest |

---

### Core Dependencies

| Library | Version |
|---------|---------|
| TensorFlow | **2.16.1** |
| Keras | **3.12.2** |
| NumPy | **1.24.3** |
| Pandas | **2.3.3** |
| Matplotlib | **3.10.9** |
| OpenCV (`opencv-python`) | **4.13.0.92** |
| Scikit-learn | **1.7.2** |
| Pillow | **12.2.0** |
| tqdm | **4.67.3** |

---

## 📦 Installation

Install the required packages using:

```bash
pip install -r requirements.txt
```

or install the core dependencies manually:

```bash
pip install tensorflow==2.16.1 keras==3.12.2 numpy==1.24.3 pandas==2.3.3 matplotlib==3.10.9 opencv-python==4.13.0.92 scikit-learn==1.7.2 pillow==12.2.0 tqdm==4.67.3
```

---

## 💡 Recommended Environment

- Python virtual environment (`venv`) or Conda is recommended.
- GPU acceleration is recommended for faster model training (CUDA and cuDNN compatible with TensorFlow 2.16.1).
